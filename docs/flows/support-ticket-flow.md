# Luồng Hỗ trợ Khách hàng (Support Ticket Flow)

1.  **Gửi ticket**: Khách hàng gặp sự cố với đơn hàng hoặc sản phẩm của Brand, điền form gửi yêu cầu hỗ trợ (Support Ticket).
2.  **Nhận thông báo**: Hệ thống thông báo đến Brand Portal.
3.  **Xử lý**: Nhân viên Brand thay đổi trạng thái ticket sang "Processing" và chat trực tiếp để giải quyết khiếu nại.
4.  **Hoàn tất**: Sau khi đổi trả hoặc giải thích thành công, ticket được chuyển sang trạng thái "Resolved" và đóng lại.
