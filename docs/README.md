# Closy Project Documentation (SmartWardrobe Backend)

## Documentation Organization Principles

- `docs/` only contains hand-written documentation drafted by humans.
- `api/swagger/` is where Swagger files automatically generated by `swaggo/swag` are stored (not included in `docs/` to avoid confusion).
- Old documents belonging to the obsolete Community Marketplace or Resale models have been moved to `docs/archive/` for storage and are not used as the development direction for the current Core MVP.

## `docs/` Directory Structure

```text
docs/
├── project/             # Vision, scope, terminology, and changelog of Closy.
├── business/            # B2B2C business model, revenue streams, and value propositions.
├── product/             # Core MVP feature list, Personas, and product roadmap.
├── domain/              # Detailed business logic for each domain (Wardrobe, AI Styling, Brand, Loyalty, Campaign, CS, Digital Sample Lab).
├── flows/               # Application flows (User flow, Brand flow, Support flow, Sample flow...).
├── system-design/       # System architecture, database, authorization, and AI Pipeline.
├── api/                 # Hand-written API design standards.
├── development/         # Local setup guide, conventions, git branch, and testing strategy.
├── deployment/          # Docker, CI/CD, VPS deploy, and monitoring guides (IP/sensitive info removed).
├── decisions/           # Architecture decision records (ADR) and major product decisions.
├── research/            # Market research and user surveys.
└── archive/             # Storage for old documents or deprecated directions.
```

## New Model Positioning: B2B2C

- **B2C**: Personal digital wardrobe application and AI outfit assistant.
- **B2B**: The project's main revenue stream through providing Brand Loyalty, Customer Service, Brand Campaign, Benefits, Insight Dashboard, and **Digital Sample Lab** services for fashion brands.
- **Non-Core Goals (Non-Goals)**: Discontinue positioning or developing Closy as a social network (Community Feed, Community Post) or a second-hand marketplace (Resale, Transfer Item, P2P transaction).
