# Sample Lab API Specs

Tài liệu thiết kế API phục vụ Digital Sample Lab.

*   `POST /api/v1/brand/samples`: Đăng tải thiết kế mẫu thử 3D/bản vẽ.
*   `POST /api/v1/brand/samples/:id/vote`: Thực hiện vote và để lại feedback cho mẫu thiết kế.
*   `GET /api/v1/brand/samples/:id/insights`: Lấy thống kê dữ liệu bình chọn ẩn danh cho nhãn hàng.
