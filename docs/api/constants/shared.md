# Hằng số Dùng chung (Shared Constants)

Các hằng số cơ bản dùng chung giữa các Module:

## 1. Tiền tệ (Currency)
*   **Đường dẫn package:** `internal/shared/domain/constants/shared/currency`
*   **Các giá trị hợp lệ:**
    *   `vnd`: Việt Nam Đồng.
    *   `usd`: Đô la Mỹ.

## 2. Giới tính (Gender)
*   **Đường dẫn package:** `internal/shared/domain/constants/shared/gender`
*   **Các giá trị hợp lệ:**
    *   `male`: Nam.
    *   `female`: Nữ.
    *   `unisex`: Phù hợp cho cả nam và nữ.

## 3. Người gửi tin nhắn (MessageSender)
*   **Đường dẫn package:** `internal/shared/domain/constants/shared/messagesender`
*   **Các giá trị hợp lệ:**
    *   `user`: Người dùng gửi tin nhắn.
    *   `ai`: Trợ lý ảo AI gửi tin nhắn.
